# Exploratory Data Analysis (EDA) Project

##  Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on a real-world dataset to uncover patterns, detect anomalies, and generate meaningful insights. 

The goal of this analysis is to understand the data structure, identify trends, and support data-driven decision-making.

---

##  Objectives
- Understand dataset structure and variables
- Handle missing values and duplicates
- Perform statistical analysis
- Identify correlations between features
- Visualize trends and distributions

---

##  Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

##  Key Steps Performed
1. Data Cleaning
2. Handling Missing Values
3. Removing Duplicates
4. Statistical Summary
5. Univariate Analysis
6. Bivariate Analysis
7. Correlation Analysis
